Forms

Forms are a part of almost any modern web application. Users need clean and easy to use forms to submit their information.

Example usage:
<form action="#" method="get">
<label>Example Label</label>
<p><input placeholder="placeholder text" /></p>
<button type="button">Submit</button>
</form>

Form Elements

GroundworkCSS makes creating beautiful forms pretty darn easy. By default, form elements such as <input>, <select> and <textarea> expand to 100% width of their containers. This makes setting up forms using the grid system really simple.
