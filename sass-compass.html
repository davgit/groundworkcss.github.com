Sass + Compass

Sass is a powerful extension to CSS and provides variables, functions, the use of mixins and extends which speeds up development and makes code cleaner and much more organized.

NOTE: For those that aren't using a CSS preprocessor yet, have no fear! We've packaged all of the normal CSS with GroundworkCSS as well. However, we strongly encourage you tolearn and use Sass. The SCSS syntax is very familiar, is easy to learn and can drastically increase your efficiency.

P.S. — If you're not a fan of the terminal, give CodeKit a try.
