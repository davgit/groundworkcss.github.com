Layout Helpers

Layout helpers work with the grid system and other elements allowing you to have a standized method for padding, positioning, aligning elements and more.


Example usage:
<div class="row">
  <div class="one half padded align-right">this column is padded and aligned right</div>
  <div class="one half double padded"><span class="callout pull-right">2nd</span> this column is double padded and has a callout element floating to the right</div>
</div>
Result:
this column is padded and aligned right
2nd this column is double padded and has a callout element floating to the right
Example usage:
<div class="callout">
  <p>This is a normal callout with padding</p>
</div>
<div class="callout zero align-center">
  <p>This is a callout with the zero and align-center helpers</p>
</div>
Result:

This is a normal callout

This is a callout with zero and align-center
