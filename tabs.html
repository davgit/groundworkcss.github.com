Tabs

Tabs offer a way for a user to easily swap between seperated content.

NEW: Aria role support added for tabs. Tabs can be created by addingclass="tabs"to a container with an unordered list and division as before, but can alternatively be done using only Aria roles to define the components. See the final example usage at the bottom of this page.
